# Box Shadow

Used to set box shadow attributes.

## Hover state
   Change the amount of box shadow on an element on hover. The syntax is : `.hover:elevation-{value}`.

### Example
<Sandbox>
  <Sandbox.Preview>
    <div className="d-inline-block ta-center flex flex-start flex-between flex-wrap" style={{
        width: '600px'
    }}>
      <div className="bg-grey-05 hover:elevation-4 flex flex-middle p-4 mx-4 my-4">Box Shadow 4 on Hover</div>
      </div>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="hover:elevation-4">Box Shadow 4</div>
  </Sandbox.Code>
</Sandbox>
