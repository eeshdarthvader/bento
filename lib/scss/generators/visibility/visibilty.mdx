# Visibilty

Utility classes to toggle the visibilty of an element


## Class reference
<Reference>
    <Property>
        <Name>.v-hidden</Name>
        <Output>visibilty: hidden;</Output>
    </Property>
    <Property>
        <Name>.v-visibile</Name>
        <Output>visibilty: visibile;</Output>
    </Property>
  </Reference>



## Example

Note that `v-hidden` hides an element, but still maintains its place in the DOM.

<Sandbox>
  <Sandbox.Preview>
    <div className="flex flex-middle flex-center bg-tertiary-100">
      <div className="bg-tertiary-500 c-white h-11 w-80p v-visible" />
    </div>
  </Sandbox.Preview>
  <Spacer my={1} />
  <Sandbox.Preview>

    </div>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="v-visible">...</div>
    <div className="v-hidden">...</div>
  </Sandbox.Code>
</Sandbox>
