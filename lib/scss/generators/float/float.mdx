# Float

Float and clearfix classes to help position elements.


## Class reference
<Reference>
    <Property>
        <Name>.c-left</Name>
        <Output>float: left;</Output>
    </Property>
    <Property>
        <Name>.f-right</Name>
        <Output>float: right;</Output>
    </Property>
    <Property>
        <Name>.f-none</Name>
        <Output>float: none;</Output>
    </Property>
    <Property>
        <Name>.clearfix</Name>
        <Text>Clears the children floats</Text>
    </Property>
</Reference>



## Float left

`.f-left` floats children within a container to its left.

<Sandbox>
  <Sandbox.Preview>
    <div className="d-inline-block h-11 bg-secondary-100 w-100p clearfix ta-center">
      <span className="f-left">Hello</span>
    </div>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="f-left">...</div>
  </Sandbox.Code>
</Sandbox>

## Float none
