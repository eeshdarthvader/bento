# Typography

Utility classes for all typography elements.

## Font family

Bento uses three font stacks. Use these classes `ff-sans-serif`, `ff-serif` and `ff-mono` to change the `font-family`

<Spacer my={8} />

<Sandbox>
  <Sandbox.Preview>
    <span className="ff-sans-serif fs-body-3 d-block">Millimeter</span>
    <span className="ff-serif fs-body-3 d-block">Millimeter</span>
    <span className="ff-mono fs-body-3 d-block">Millimeter</span>
  </Sandbox.Preview>
  <Sandbox.Code>
    <p className="ff-sans-serif">...</p>
    <p className="ff-serif">...</p>
    <p className="ff-mono">...</p>
  </Sandbox.Code>
</Sandbox>

  </Property>
  <Property>
    <Name>.fs-body-2</Name>
    <Output>font-size:16px;</Output>
  </Property>
  <Property>
    <Name>.fs-body-3</Name>

## Font weight

The available range of weights is `100`(Ultra thin) to `900`(Black).
Some of the commonly used weights are `.fw-400`, `.fw-500`, `.fw-600` and `.fw-800`

Additionally, `.lh-none` sets the `line-height` to `0`. It's best to not use it unless needed for a specific usecase.

## Font style

This property allows you to make text appear italicized. By default the `font-style` is set to `normal`

  <Property>

  </Property>
  <Property>
