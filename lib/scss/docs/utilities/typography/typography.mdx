# Typography

Utility classes for all typography elements.

## Font size

Bento's type scale has the following classes to change the font size of an element. Use it in combination with font weight and line height for better harmony.

<Reference>
  <Property>
    <Name>.fs-caption</Name>
    <Output>font-size:10px;</Output>
  </Property>
  <Property>
    <Name>.fs-caption-2</Name>
    <Output>font-size:12px;</Output>
  </Property>
  <Property>
    <Name>.fs-body</Name>
    <Output>font-size:14px;</Output>
  </Property>
  <Property>
    <Name>.fs-body-2</Name>
    <Output>font-size:16px;</Output>
  </Property>
  <Property>
    <Name>.fs-body-3</Name>
    <Output>font-size:20px;</Output>
  </Property>
  <Property>
    <Name>.fs-heading</Name>
    <Output>font-size:24px;</Output>
  </Property>
  <Property>
    <Name>.fs-heading-2</Name>
    <Output>font-size:32px;</Output>
  </Property>
  <Property>
    <Name>.fs-heading-3</Name>
    <Output>font-size:36px;</Output>
  </Property>
  <Property>
    <Name>.fs-display</Name>
    <Output>font-size:48px;</Output>
  </Property>
</Reference>

## Font weight

The available range of weights is `100`(Ultra thin) to `900`(Black).
Some of the commonly used weights are `.fw-400`, `.fw-500`, `.fw-600` and `.fw-800`

## Line height

Change the line height density with these classes.

<Reference>

  <Property>
    <Name>.lh-solid</Name>
    <Output>line-height:1;</Output>
  </Property>
  <Property>
    <Name>.lh-title</Name>
    <Output>line-height:1.25;</Output>
  </Property>
  <Property>
    <Name>.lh-copy</Name>
    <Output>line-height:1.5;</Output>
  </Property>
  </Reference>
