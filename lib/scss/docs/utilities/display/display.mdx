## Inline Block

`.d-inline-block` creates inline elements which behave as blocks.

<Sandbox>
  <Sandbox.Preview>
    <div className="bg-tertiary-400 c-white d-inline-block mr-3">
      <div className="flex flex-middle flex-center h-40">
        This only
      </div>
    </div>

    <div className="bg-tertiary-400 c-white d-inline-block mr-3">
      <div className="flex flex-middle flex-center h-40">
        takes as much space
      </div>
    </div>

    <div className="bg-tertiary-400 c-white d-inline-block">
      <div className="flex flex-middle flex-center h-40">
        as it needs
      </div>
    </div>
  </Sandbox.Preview>

</Sandbox>
