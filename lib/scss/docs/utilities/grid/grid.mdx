# Grid

The grid system provides us with a device agnostic layout system.



## Class reference

<Reference>
    <Property>
        <Name>.container</Name>
        <Text>Sets width and centers outermost div</Text>
    </Property>
    <Property>
        <Name>.row</Name>
        <Text>Aligns children columns</Text>
    </Property>
    <Property>
        <Name>.col</Name>
        <Text>Set width of individual column</Text>
    </Property>
</Reference>



## Grid construction

A basic grid is constructed out of three elements, `.container`, `.row` and `.col`

<Spacer my={8} />

<Sandbox>
  <Sandbox.Preview>
    <Container>
      <Row>
        <Col>
          <Cell>col-12</Cell>
        </Col>
        <Col>
          <Cell>col-12</Cell>
        </Col>
      </Row>
    </Container>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="container">
      <div className="row">
        <div className="col-12">...</div>
        <div className="col-12">...</div>
      </div>
    </div>
  </Sandbox.Code>
</Sandbox>

## Offset columns

It's possible to leave some some columns empty. The syntax for that is `col-offest-{number}`

<Sandbox>
  <Sandbox.Preview>
    <Container>
      <Row>
        <Col span={8}>
          <Cell>col-8</Cell>
        </Col>
        <Col span={12} offset={4}>
          <Cell>col-12 (offset by 4 columns)</Cell>
        </Col>
      </Row>
    </Container>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="container">
      <div className="row">
        <div className="col-8">...</div>
        <div className="col-12 col-offset-4">...</div>
      </div>
    </div>
  </Sandbox.Code>
</Sandbox>
<Spacer my={16} />

## Distribution

## Ordering

It possible to alter the order of the colmumns using the `last` or `fist` classes. Notice that these classes are applied on the `col` element.

<Sandbox>
  <Sandbox.Preview>
    <Container>
      <Row>
        <Col span={8}>
          <Cell>1</Cell>
        </Col>
        <Col span={8}>
          <Cell>2</Cell>
        </Col>
        <Col span={8} first>
          <Cell dark>3</Cell>
        </Col>
      </Row>
    </Container>
  </Sandbox.Preview>
  <Spacer my={4} />
  <Sandbox.Preview>
    <Container>
      <Row>
        <Col span={8} last>
          <Cell dark>1</Cell>
        </Col>
        <Col span={8}>
          <Cell>2</Cell>
        </Col>
        <Col span={8}>
          <Cell>3</Cell>
        </Col>
      </Row>
    </Container>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="container">
      <div className="row">
        <div className="col-8">1</div>
        <div className="col-8">2</div>
        <div className="col-8 first">3</div>
      </div>
      <div className="row">
        <div className="col-8 last">1</div>
        <div className="col-8">2</div>
        <div className="col-8">3</div>
      </div>
    </div>
  </Sandbox.Code>
</Sandbox>

## Reversing

You can reverse the order of the columns by adding a `reverse` class to the `row`.

<Sandbox>
  <Sandbox.Preview>
    <Container>
      <Row reverse>
        <Col span={8}>
          <Cell>1</Cell>
        </Col>
        <Col span={8}>
          <Cell>2</Cell>
        </Col>
        <Col span={8}>
          <Cell>3</Cell>
        </Col>
      </Row>
    </Container>
  </Sandbox.Preview>
  <Sandbox.Code>
    <div className="container">
      <div className="row reverse">
        <div className="col-8">1</div>
        <div className="col-8">2</div>
        <div className="col-8">3</div>
      </div>
    </div>
  </Sandbox.Code>
</Sandbox>
