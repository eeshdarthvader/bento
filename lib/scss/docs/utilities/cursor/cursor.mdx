## Auto

`.c-auto` changes the cursor depending on the context of the content being hovered on. Example:- cursor becomes a select on hovering over text.

  <Sandbox.Preview>
    <div className="d-inline-block h-40 bg-secondary-500 ta-center flex flex-center flex-middle">

    </div>

  <Sandbox.Code>
    <div className="c-auto">...</div>
  </Sandbox.Code>
</Sandbox>
