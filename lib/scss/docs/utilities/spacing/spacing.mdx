# Spacing

Defining the units for spacing values

## Custom scale

Our spacing unit is derived from our 4px baseline grid. To 
establish consistency across various components, we defined units of spacing,
abstracting away the technical details. 

Spacing units are set in multiples of 4 i.e. 1 unit = 4px
